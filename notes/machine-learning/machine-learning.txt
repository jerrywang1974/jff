## Deep Learning

https://en.wikipedia.org/wiki/Comparison_of_deep_learning_software

Graphical visualization:    DIGITS, TensorBoard
Machine learning workflow management (Dataset preparation, save/load training loop): Keras, TensorFlow slim
Computational graph(CG) management (Build/optimize CGs, forward/back propagation): Theano, TensorFlow, Torch.nn
Multi-dimensional array processing (High-level array manipulation): NumPy, CuPy, Eigen, Torch core, libgpuarray
Numerical computation (Maxtrix operation, convolution): BLAS(OpenBLAS, MKL, AMD Core Math Library - ACML, Apple Accelerate Framework), cuBLAS, cuRAND, cuDNN, MKL DNN, CUDA, OpenCL CUDAToolkit
Computational device: CPU, GPU, TPU, FPGA
